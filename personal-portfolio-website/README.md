# 🌐 My First Personal Portfolio Website

Welcome to my personal portfolio website repository! This is a clean, modern, and responsive single-page web application designed to showcase my skills, projects, and educational background as a Full-Stack Developer.

Built under the brand name **Krishi Code**, this portfolio acts as my digital resume for recruiters and clients.

---

## 📸 Design Blueprint & Layout
The layout is designed with a professional dark slate theme, structured into the following primary sections:
1. **Navigation Bar:** Sticky navigation for smooth layout scrolling.
2. **Hero Section:** Introduction with a custom developer avatar.
3. **About Me:** Summary of my academic background (MCA) and goals.
4. **Skills Grid:** Visual representation of core technical skills.
5. **Projects Section:** Grid displaying 8 wireframe/live project cards.
6. **Contact Form:** Interactive form for inquiries.
7. **Footer:** Quick links and copyright metadata.

---

## 🛠️ Tech Stack Used

- **HTML5:** Semantic structural layout tags.
- **CSS3:** Custom Vanilla CSS styling, flexbox/grid layout systems, and responsive design properties.
- **FontAwesome Icons:** For clean developer-centric technology icons.
